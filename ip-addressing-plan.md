# IP Addressing Plan

The network uses private IPv4 addressing.

10.11.0.0/16 network
Used for internal user networks.

20.12.0.0/16 network
Used for infrastructure and server networks.

Example subnets:

10.11.10.0/24 — US Users  
10.11.20.0/24 — US Management  
10.11.30.0/24 — Canada Users  
10.11.40.0/24 — Canada Management  

20.12.10.0/24 — Data Center Servers  
20.12.20.0/24 — ISP connectivity
