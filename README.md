# Enterprise Multi-Site Network Architecture

## Overview

This project demonstrates the design and implementation of a multi-site enterprise network using Cisco Packet Tracer. The network connects two company locations in the United States and Canada and includes data center infrastructure, VLAN segmentation, dynamic routing, and external services.

The objective of this project was to design a highly available and scalable enterprise network capable of supporting internal employees and externally hosted services.

## Network Architecture

The network is divided into multiple major sections:

United States Site  
Canada Site  
Service Provider / ISP Network  
Enterprise Data Centers  

Each site contains multiple routers, switches, servers, and end-user devices.

## Key Technologies Used

Cisco Packet Tracer

## Network Features

Multi-site enterprise architecture  
Multiple routers and switches  
VLAN segmentation for network isolation  
Wireless access for employees  
Internal services such as DHCP  
External services including DNS, Email, FTP, and Web servers  
ISP connectivity between sites

## VLAN Structure

Example VLAN segmentation includes:

VLAN 10 – US Users  
VLAN 20 – US Management  
VLAN 30 – Canada Users  
VLAN 40 – Canada Management  

This segmentation improves security and reduces broadcast traffic.

## Services Implemented

DNS server  
Email server  
FTP server  
Web hosting servers  

These services simulate a company hosting services for external customers.
