# Network Architecture

This network follows a hierarchical enterprise network design consisting of multiple layers:

Core Layer
Handles high-speed routing between major network segments and connects the US and Canada sites.

Distribution Layer
Aggregates access switches and enforces routing policies and segmentation.

Access Layer
Connects end-user devices such as PCs and wireless clients.

Data Center Layer
Hosts internal and external services including DNS, FTP, Email, and Web servers.

ISP Layer
Simulates external internet connectivity between enterprise sites.
